== Description ==

We have combined the capabilities of leading Marketing Tool so that our users don't have to use multiple plugins to generate more leads, acquire new customers, optimize user on-boarding, boost customer retention, collect visitor feedback and earn referrals; everything is available for one plugin, and can be setup in minutes.

Benefits:

*  Increase Website Sales
*  Collect Emails & Reduce Cart Abandonment
*  Schedule Sales Calls
*  Increase Website Traffic
*  Reduce Subscription Cancellations
*  Increase Customer Referrals & Social Shares All Features
*  Real-Time Social Proof
*  Exit Intent Emails & Surveys
*  Welcome & Behaviour Driven Messages
*  Welcome Bar
*  Cookie Policy Notification
*  Click to Call Widget and many, many more...

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/notifia` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Select 'Notifia' in the left sidebar. If you already have a Notifia account, click on the 'Log in' link. If you are new to Notifia, click Sign Up.
